# Venkatesh Portfolio

This project is now scaffolded as a minimal Next.js app using `venky.js` as the main page component.

## Setup

1. Install Node.js 18+ on your machine.
2. Run:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open: `http://localhost:3000`

## Notes

- `app/page.js` loads `venky.js` as the portfolio page.
- `app/globals.css` contains the CSS classes used by the component.
- If you want to keep the plain HTML version, `por.html` and `style.css` remain unchanged.
